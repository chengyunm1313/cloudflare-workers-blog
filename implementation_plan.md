# 繁體中文翻譯任務

## 目標

將專案中所有的簡體中文翻譯成台灣繁體中文。

## 範圍

- `README.md`
- `index.js`
- `themes/` 下各主題的 HTML 檔案

## 任務拆解

1. 轉換 `README.md` 中的文字
2. 轉換 `index.js` 中的文字與註解
3. 轉換 `themes/default/` 中的所有 HTML 檔案
4. 轉換其他主題（如 `JustNews`）中的 HTML 檔案

## 驗收標準

- 所有使用者可見文字皆為台灣繁體中文
- 程式碼邏輯保持完整
- 註解與文件符合台灣用語習慣

## 風險與回滾策略

- 風險：翻譯錯誤導致程式碼報錯（如 JSON 鍵名被改動）
- 回滾：使用 Git checkout 回滾特定檔案
